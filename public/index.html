Exercise tracker

Create a new user

POST /api/exercise/new-user


Add exercises

POST /api/exercise/add


GET users's exercise log:
GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example Usage:

[this_project_url]/api/exercise/log?userId="XXXX"
[this_project_url]/api/exercise/log?userId="XXXX"&from="1994-03-21"
[this_project_url]/api/exercise/log?userId="XXXX"&from="xxxx-xx-xx"
[this_project_url]/api/exercise/log?userId="XXXX"&from="xxxx-xx-xx"&limit="2"
