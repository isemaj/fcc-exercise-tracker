Exercise tracker

Create a new user

POST /api/exercise/new-user


Add exercises

POST /api/exercise/add


GET users's exercise log:
GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example Usage:

[this_project_url]/api/exercise/log?userId=9EQ03yfme
[this_project_url]/api/exercise/log?userId=9EQ03yfme&from=2004-03-01
[this_project_url]/api/exercise/log?userId=9EQ03yfme&from=2004-03-01&to=2006-11-31
[this_project_url]/api/exercise/log?userId=9EQ03yfme&from=2004-03-01&to=2006-11-31&limit=2
